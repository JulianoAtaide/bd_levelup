##Sistema de Gestão para Lan House
#Visão Geral do Projeto
Este repositório contém o Modelo Lógico, os scripts DDL (Data Definition Language) e DML (Data Manipulation Language) desenvolvidos para a Lan House fictícia "Levelup".
Este projeto foi criado como parte de uma disciplina de Modelagem e Banco de Dados em nível universitário, com o objetivo de aplicar e consolidar os conceitos de Normalização (1FN, 2FN, 3FN), Integridade Referencial e Linguagem SQL em um cenário prático.
